# Bookmark Organizer Demo

这个 demo 用于展示 Chrome 书签整理的完整流程，包括导入/导出、可达性检测、AI 整理、去重合并、标签管理与报告导出。

## 功能概览
- 导入/导出：支持 Chrome HTML 与 JSON
- 可达性检测：后端路由批量检测 URL 状态与耗时
- 元数据抓取：获取页面标题与 favicon
- AI 整理：生成目录结构、标签与重命名建议
- 去重合并：按 URL 合并重复书签
- 失效清理：分组并批量删除失效链接
- 报告导出：CSV/JSON（包含状态码与响应时间）

## 使用说明
1. 点击“导入书签”上传 HTML 或 JSON 文件
2. 可选：执行可达性检测与元数据抓取
3. 点击“AI 整理”生成标签与目录建议，再“应用 AI 建议”
4. 使用搜索与批量编辑进行进一步整理
5. 通过标签面板进行合并/移除
6. 导出 HTML/JSON 或报告文件

## 并发设置
- 可达性检测与元数据抓取支持并发数配置
- 默认 8，最大 32

## 数据持久化
- 优先使用 IndexedDB
- 不可用时自动降级到 localStorage

## AI 配置
- 依赖 `DEEPSEEK_API_KEY` 环境变量
- 使用 deepseek 模型进行结构化输出
